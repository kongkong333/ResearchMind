# 研究报告

## 一、研究概览
主题：RL
论文数量：2

## 二、重点论文解读
1. When RL Fails after SFT: Rejuvenating Model Plasticity for Robust SFT-to-RL Handoff（当 RL 在 SFT 之后失败时：恢复模型可塑性以实现稳健的 SFT 到 RL 切换）
简要概述：本文针对过度SFT导致模型可塑性丧失进而使后续RL失效的问题，提出结合基础锚定模型融合与目标神经元重置的Rejuvenation方法，有效恢复了模型可塑性并显著提升了RL性能与泛化能力。
针对的问题：过度SFT会导致模型丧失可塑性（表现为过度自信的token分布和尖锐的参数景观），使得后续RL阶段难以有效优化模型。
方法：提出Rejuvenation方法，通过基础锚定模型融合减少SFT引起的参数漂移，并结合目标神经元重置缓解模型僵化。
创新点：揭示了过度SFT导致模型可塑性丧失的内在机制，并提出结合模型融合与神经元重置的机制以恢复可塑性。
结果：在数学推理和智能体任务上，一致提升了过度SFT模型的RL性能，并增强了对分布外任务的泛化能力。
局限：摘要未明确提及，但目标神经元重置的具体选择标准及在更多复杂任务上的有效性有待进一步验证。

2. Breaking Entropy Bounds: Accelerating RL Training via MTP with Rejection Sampling（打破熵界限：通过 MTP 和拒绝采样加速 RL 训练）
简要概述：本文提出Bebop系统，通过引入概率拒绝采样和端到端TV损失优化多令牌预测(MTP)，解决了大模型RL训练中MTP接受率下降的问题，在Qwen系列模型上实现了最高1.8倍的端到端加速。
针对的问题：大模型强化学习(RL)训练中Rollout阶段是瓶颈，而多令牌预测(MTP)在RL阶段因模型熵增导致接受率显著下降，加速效果受限。
方法：采用概率拒绝采样替代贪婪采样以缓解熵干扰；提出端到端TV损失直接优化多步拒绝采样接受率；在RL前进行MTP预训练以避免昂贵的在线更新。
创新点：揭示了MTP接受率与RL阶段模型熵增的负线性关系；提出端到端TV损失直接优化接受率；证明了预训练MTP结合拒绝采样可免除在线更新的必要性。
结果：接受率提升约10%（最高达95%），推理吞吐量提升最高25%，在Qwen3.5/3.6/3.7异步RL训练中实现最高1.8倍端到端加速。
局限：摘要未明确提及局限性，但端到端TV损失可能增加训练计算开销，且实验主要基于Qwen系列模型验证。

## 三、研究机会
1. 这些论文可利用/结合之处（如模型结构、方法论、技术路线等）
- 探索更动态的SFT-RL过渡策略、设计无需神经元重置的可塑性保持正则化方法，或将其扩展至多模态及更复杂的RLHF流程中。
- 探索动态适应RL过程中熵变化的MTP架构；将端到端TV损失和拒绝采样机制扩展至更多RL算法及不同架构的大模型中。
2. 这些论文尚未做好的地方
- 现有研究缺乏对SFT到RL交接阶段模型可塑性丧失现象的系统分析，且缺少在保留SFT先验的同时恢复模型可塑性的有效机制。
- 传统MTP训练目标（交叉熵或KL）在RL高熵环境下表现次优，缺乏对RL阶段MTP接受率下降机制的系统性分析及针对性优化方案。
- 摘要未明确提及，但目标神经元重置的具体选择标准及在更多复杂任务上的有效性有待进一步验证。
- 摘要未明确提及局限性，但端到端TV损失可能增加训练计算开销，且实验主要基于Qwen系列模型验证。
